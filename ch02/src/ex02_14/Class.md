> 

# 클래스란?

객체를 정의하는 틀 또는 설계도와 같은 의미로 사용됩니다.

자바에서는 이러한 설계도인 클래스를 가지고, 여러 객체를 생성하여 사용하게 됩니다.

클래스는 객체의 상태를 나타내는 필드(field)와 객체의 행동을 나타내는 메소드(method)로 구성됩니다.

즉, 필드(field)란 클래스에 포함된 변수(variable)를 의미합니다.

또한, 메소드(method)란 어떠한 특정 작업을 수행하기 위한 명령문의 집합이라 할 수 있습니다.

## 클래스 선언

간단한 구조의 클래스를 선언하는 예는 다음과 같습니다.

```java
// 클래스 CAR
class CAR{
	String modelName;	// modelName - String 형 필드
	int modelYear;		// modelYear - int 형 필드
	String color;
	int maxSpeed;
}
```

CAR은 클래스 이름입니다. 클래스 안에는 4개의 데이터 요소를 가지고 있습니;다.

데이터 요소를 필드(field)라고 합니다. 4개의 필드(field)를 가지고 있는 클래스 CAR 입니다.

클래스형 변수를 사용할 때는 먼저 클래스형 변수(실체를 참조하는 변수) 를 만들고, 그리고 실체인 클래스 인스턴슬르 생성해야 합니다. 

```java
CAR car;		 		 	// CAR형의 클래스 변수 car 선언
car = new CAR();	// CAR형의 클래스 인스턴스(실체)를 생성
```

아래 처럼 선언하면 두가지를 한 번에 할 수도 있습니다.

```java
CAR car = new CAR();	// 변수와 인스턴스 생성을 한꺼번에 선언
```
